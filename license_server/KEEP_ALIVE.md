# Сервер не засыпает — Keep-Alive

На Render (бесплатный план) сервер **засыпает** через ~15 минут без обращений. После сна первый запрос может занимать 50–60 секунд (холодный старт).

## Что уже сделано

- Увеличен таймаут до 90 секунд
- Добавлен повтор при таймауте в `generate_keys_remote.py`

## Как держать сервер активным

Используй бесплатный мониторинг:

1. Зайди на **https://uptimerobot.com**
2. Создай аккаунт
3. **Add New Monitor**
   - Type: **HTTP(s)**
   - URL: `https://psylocyba-license.onrender.com/verify?key=test&hwid=test`
   - Monitoring Interval: **5 минут**
4. Сохрани

Сервер будет получать запрос каждые 5 минут и не будет засыпать.

---

**Альтернатива:** Cron-job.org — настроить задачу, которая раз в 10–15 минут открывает этот URL.
